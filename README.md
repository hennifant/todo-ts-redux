Um dieses Repository zu klonen befolgen Sie folgende Schritte:

```jsx
git@github.com:hennifant/todo-ts-redux.git
```

```jsx
cd todo-ts-redux
```

```jsx
npm install
```

```jsx
npm start
```

```jsx
npm run build
```
